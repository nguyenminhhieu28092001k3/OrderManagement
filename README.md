## 🛠️ Cài đặt

### 1. Yêu cầu hệ thống
- **Java**: Java 21 hoặc cao hơn
- **Docker**: Docker & Docker Compose (khuyến nghị)
- **Database**: PostgreSQL 15+ (được cài tự động bằng Docker)
- **IDE**: NetBeans (khuyến nghị) hoặc IntelliJ IDEA

### 2. Biên dịch và chạy ứng dụng
```bash
# Windows
.\run.bat

```
